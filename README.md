# Biohacker Alliance API

Backend Serverless Functions for Biohacker Alliance.

## Deployment

Deploy to Vercel with these environment variables:

- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY  
- SUPABASE_SERVICE_ROLE_KEY

## Endpoints

- POST /api/experts/register - Expert registration
- GET /api/experts - List approved experts
- POST /api/newsletter - Newsletter subscription
